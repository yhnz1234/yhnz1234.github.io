### 🎯 我学会了
✍️ 写文章
  ├─ 命令：hexo new post "标题"
  ├─ 位置：source/_posts/标题.md
  └─ 图片：放 source/images/，文中用 /images/xxx.jpg

🔍 预览
  ├─ hexo clean    
  ├─ hexo generate  
  └─ hexo server → http://localhost:4000

🚀 发布
  ├─ git add .          添加所有修改
  ├─ git commit -m "备注"
  ├─ git push          (源代码→src 分支)
  └─ hexo d            (网站文件→main 分支)

🌐 访问
  └─ https://yhnz1234.github.io
